Multiplayer Wordle

Create a new game or join an existing one to play Wordle with friends!

How to Play

  • Enter Your Name: Enter your name in the "Your Name" input box. This name will be displayed to other players.
  • Create a Game: Click "Create Game" to start a new Wordle game and get a unique Game ID.
  • Join a Game: Enter a Game ID in the input box and click "Join Game" to join an existing game.
  • Share the Game ID: If you create a game, share the Game ID with your friends so they can join.
  • Play Wordle: Once you're in a game, you'll play the classic Wordle game with other players.
  • Guesses are Shared: All players in the game share the same Wordle board. Your guesses will appear for everyone.
  • Win Together: The game ends when someone guesses the word correctly. Everyone in the game wins!
